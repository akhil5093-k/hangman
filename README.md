Hangman Game (Tkinter + Python)

A fully interactive Hangman Game built using Python and Tkinter, featuring multiple categories, difficulty levels, a countdown timer, visual hangman drawing, and a background image.

This project is a fun and educational way to learn GUI programming, game logic, and event handling in Python.
